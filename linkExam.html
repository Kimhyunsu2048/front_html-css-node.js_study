Anchor Element(Tag)에 대하여

a(Anchor) Tag : 링크를 적용하는 태그입니다. 대표적 속성은 href="URL" 이며, target 이라는 속성 또한 정말 많이 사용됩니다. 두 속성은 반드시 기억하세요. 아래는 예제입니다. 속성은 영어로 Attribute or Properties 라고 하는데 둘다 같이 기억하세요!! 속성은 key(속성명)="value(속성값)"으로 구성되어서 특정 요소(Element)가 수행할 일의 구체적인 값을 지정하도록 하는데 쓰여집니다. target 은 링크된 페이지의 content 가 어디에 보여질지의 속성입니다. default 값은 _self 입니다.
